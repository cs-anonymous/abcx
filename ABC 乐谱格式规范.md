## 一、什么是 ABC

ABC 是一种**基于纯 ASCII 文本的音乐记谱格式**，由 Chris Walshaw 发明。它最初用于记录和传播民间音乐（folk/traditional music），后来被广泛应用于早期音乐、古典音乐等多个领域。

**核心优势：**
- **人类可读可写**——用键盘就能打字记谱，无需专业软件
- **极小体积**——一千首曲子可以装进一张软盘
- **平台无关**——不依赖任何操作系统或专用硬件
- **LLM 友好**——纯文本、高压缩率、天然编码音乐重复与结构
- **可转换**——可转为 MIDI、MusicXML、标准五线谱（PNG/PDF）

**主要局限：**
- 不是完整的音乐记谱——最适合单旋律乐器
- 软件来源不统一，存在实现差异
- 不记录页面排版信息

## 二、基础知识

### 2.1 音符表示

ABC 用字母 `A` 到 `G` 表示音名，大小写区分八度：

| 字母               | 音高位置      | 说明            |
| ---------------- | --------- | ------------- |
| `CDEFGAB`        | 小字一组（中央 C 所在八度） | 大写字母，C = 中央 C（Middle C） |
| `cdefgab`        | 小字二组 | 小写字母 |
| `C,D,E,F,G,A,B,` | 小字组 | 逗号表示降一个八度 |
| `c'd'e'f'g'a'b'` | 小字三组 | 撇号表示升一个八度 |

> **注意**：ABC 的默认音域（`C` 到 `b`）对应的是**小字一组**到**小字二组**（约等于钢琴上的 C4–B5），这是民间音乐最常用的音域。更低或更高的音通过追加逗号/撇号表示。

### 与国际标准音高记号的对应

标准钢琴有 88 个键（A₀–C₈，约七个八度加一个小三度），中文教学体系称之为**大字组/小字组体系**。该体系不仅分组，**每个音都有独立的字母+上标/下标标记**。下表列出完整对照：

| 中文分组   | 中文音名标记        | 科学音高      | 钢琴键     | ABC 写法           | 说明                       |
| ------ | ------------- | --------- | ------- | ---------------- | ------------------------ |
| 大字二组   | `A₂` `B₂`       | A₀–B₀     | #1–#2   | `A,,,,`–`B,,,,`    | 大写字母+下标 2                 |
| 大字一组   | `C₁`–`B₁`      | C₁–B₁     | #3–#14  | `C,,,`–`B,,,`      | 大写字母+下标 1                 |
| 大字组    | `C`–`B`        | C₂–B₂     | #15–#26 | `C,,`–`B,,`        | 大写字母（无标记）                 |
| 小字组    | `c`–`b`        | C₃–B₃     | #27–#38 | `C,`–`B,`          | 小写字母（无标记）                 |
| **小字一组** | **`c¹`–`b¹`** | **C₄–B₄** | **#39–#50** | **`C`–`B`**       | **小写字母+上标 1，`c¹` = 中央 C** |
| **小字二组** | **`c²`–`b²`** | **C₅–B₅** | **#51–#62** | **`c`–`b`**       | **小写字母+上标 2**             |
| 小字三组   | `c³`–`b³`      | C₆–B₆     | #63–#74 | `c'`–`b'`          | 小写字母+上标 3                 |
| 小字四组   | `c⁴`–`b⁴`      | C₇–B₇     | #75–#86 | `c''`–`b''`        | 小写字母+上标 4                 |
| 小字五组   | `c⁵`          | C₈        | #87     | `c'''`             | 小写字母+上标 5，钢琴最高音           |

> **关键规律**：中文体系以"组名→标记"的映射为：
> - **大字**组用**大写字母**，数字越往下标越大（大字二组 > 大字一组 > 大字组）
> - **小字**组用**小写字母**，数字越往上标越大（小字组 → 小字一组 c¹ → 小字二组 c² …）
> - 大字组和小字组都**无数字标记**，大小写区分高低
> 
> ABC 的逗号/撇号以中央 C 为基准逐层上下扩展。大写 `C` = 中央 C = `c¹` = C₄，加一个逗号降到 C₃（`C,` = 小字组），加撇号升到 C₆（`c'` = 小字三组）。

**ABC 音域示例（从小字组到小字四组）：**

```
C, D, E, F, G, A, B,           ← 小字组（C₃–B₃）
C D E F G A B                   ← 小字一组（C₄–B₄，中央 C 所在）
c d e f g a b                   ← 小字二组（C₅–B₅）
c' d' e' f' g' a' b'           ← 小字三组（C₆–B₆）
c'' d'' e'' f'' g'' a'' b''    ← 小字四组（C₇–B₇）
```

> 注：`C` 到 `b` 之间只有两个八度，这是 ABC 的核心音域。低于小字组（`C,`）的音在民间音乐中很少使用，如需记谱则需追加逗号（`C,,` = C₂ 大字组）。

**C 大调音阶示例（从中央 C 上行两个八度）：**

```
CDEFGABcdefgabc'
```

> 注意：音域可以进一步扩展——添加更多逗号（向下）或撇号（向上）即可。不过民间音乐的记谱很少超出小字一组到小字三组（`C` 到 `b'`）这个范围。

### 2.2 音符时值

通过头部字段 `L:` 设置**默认音符长度**：

| `L:` 值   | 英文术语       | 中文    | 常见用途                  |
| -------- | ---------- | ----- | --------------------- |
| `L:1/2`  | Minim      | 二分音符  | —                     |
| `L:1/4`  | Crotchet   | 四分音符  | 华尔兹                   |
| `L:1/8`  | Quaver     | 八分音符  | Jig, Reel, Polka（最常用） |
| `L:1/16` | Semiquaver | 十六分音符 | 快速装饰                  |

**修改单个音符时值：**

| 写法 | 含义 | 示例 |
|------|------|------|
| `C` | 默认时值 | 若 `L:1/8`，则为八分音符 |
| `C/` 或 `C/2` | 默认时值的一半 | 十六分音符（当 L:1/8）|
| `C/4` | 默认时值的 1/4 | 三十二分音符 |
| `C2` | 默认时值的 2 倍 | 四分音符（当 L:1/8）|
| `C3` | 默认时值的 3 倍 | 附点四分音符 |
| `C4` | 默认时值的 4 倍 | 二分音符（当 L:1/8）|
| `C3/2` | 默认时值的 1.5 倍 | 附点八分音符 |

**节奏快捷符号 `>` 和 `<`：**

| 写法     | 等价         | 说明                      |
| ------ | ---------- | ----------------------- |
| `A>B`  | `A3/2B1/2` | 前长后短（类似附点节奏）            |
| `A<B`  | `A1/2B3/2` | 前短后长（苏格兰 Strathspey 节奏） |
| `A>>B` | 双附点        | 更极端的长短比                 |

**Hornpipe 节奏对比：**

```abc
L:1/16
D3EF3G        ← 用十六分音符写

L:1/8
D>EF>G        ← 用 > 符号简写，更清晰
```

### 2.3 休止符

用小写字母 `z` 表示，时值规则与音符相同：

| 写法 | 含义 |
|------|------|
| `z` | 一个默认时值的休止 |
| `z2` | 两倍默认时值 |
| `z/` | 一半默认时值 |
| `z4` | 四倍默认时值（全休止符级别）|

### 2.4 升号、降号与还原号

| 符号 | 前缀 | 示例 | 说明 |
|------|------|------|------|
| 升号 | `^` | `^c` | C sharp |
| 降号 | `_` | `_B` | B flat |
| 还原号 | `=` | `=c` | C natural |
| 重升号 | `^^` | `^^f` | F double sharp |
| 重降号 | `__` | `__b` | B double flat |

**G 大调音阶：** `GABcde^fg`
**G 小调音阶：** `GA_Bcd_efg`

### 2.5 调号（K: 字段）

调号用 `K:` 字段指定，放在头部最后一行：

```abc
K:C       ← C 大调（无升降号）
K:G       ← G 大调（一个升号）
K:Gm      ← G 小调（m 或 min）
K:Bb      ← Bb 大调
K:C#      ← C# 大调
K:Edor    ← E Dorian 调式
K:AMix    ← A Mixolydian 调式
K:Gmixolydian  ← 完整写法也行
K:HP      ← 苏格兰风笛调（无调号，暗示风笛音阶）
K:A =C    ← A 大调但 C 还原（复杂调号）
```

**调式写法**（全名或前三个字母均可，空格和大小写不敏感）：
- `K:G mix` / `K:Gmix` / `K:G mixolydian` / `K:Gmixolydian` 都是正确的

**模态调式速查：**

| 模态          | 含义            |
| ----------- | ------------- |
| `maj` / 无后缀 | 大调            |
| `m` / `min` | 小调            |
| `dor`       | Dorian        |
| `mix`       | Mixolydian    |
| `aeo`       | Aeolian（自然小调） |
| `ion`       | Ionian（同大调）   |
| `lyd`       | Lydian        |
| `phr`       | Phrygian      |
| `loc`       | Locrian       |

### 2.6 拍号（M: 字段）

```abc
M:4/4     ← 四四拍
M:3/4     ← 三四拍（华尔兹）
M:6/8     ← 六八拍（Jig）
M:2/4     ← 二四拍
M:C       ← 普通拍号（Common time = 4/4）
M:C|      ← 切割拍号（Cut time = 2/2）
```

**节奏字段 `R:`**（用于分类和检索，不影响演奏）：

```abc
R:Jig
R:Reel
R:Waltz
R:Polka
R:Schottische
R:Bourree
```

## 三、进阶语法

### 3.1 小节线与反复记号

- **`|`** — 普通小节线
- **`||`** — 双小节线
- **`|]`** — 细-粗双小节线（段落结束）
- **`[|`** — 粗-细双小节线
- **`|:`** — 反复开始
- **`:|`** — 反复结束
- **`::`** — `:||:` 的简写（一段结束同时另一段开始）
- **`|1 ... :|`** — 第一结尾
- **`|2 ... |]`** — 第二结尾

**书写要点：**
- 小节线和数字之间**不能有空格**——`|1` 和 `[1` 是正确的，`| 1` 和 `[ 1` 是错误的
- `[1` 可以省略为 `|1`（省略方括号）

**示例：**

```abc
M:6/8
L:1/8
R:Jig
K:G
GAB cde | fgf edc | BAG
```

**带反复的写法：**

```abc
DE FF |1 GA Bc :|2 GA BG ||
```

也可以省略方括号中的 `[`：

```abc
DE FF | GA Bc :|| GA BG ||
```

> 注意：小节线和数字之间**不能有空格**——`|1` 和 `[1` 是正确的，`| 1` 和 `[ 1` 是错误的。

### 3.2 装饰音与倚音

**通用装饰音符号 `~`：**

```abc
~G2          ← G 加装饰音（通常解释为转音或滚奏）
```

**精确装饰音/倚音（花括号 `{}`）：**

```abc
{GAGDG}G2    ← G 前面加五个精确的装饰音符
{g}a         ← 在 a 前面加一个 g 倚音
```

花括号内的音符**没有固定时值**，所以不能使用 `G2` 或 `G/` 这样的时值修饰符。

### 3.3 连音线与圆滑线

| 符号    | 含义                        |
| ----- | ------------------------- |
| `-`   | **连音线**（Tie）——连接两个相同音高的音符 |
| `( )` | **圆滑线**（Slur）——将多个音符连奏    |


```abc
|DEF-|FGA|           ← Tie 跨小节
|DEF-FGA|            ← Tie 在同一小节内
(DEF|GAB)            ← Slur 跨多个音符
(^G A B/c/|E4 D4)    ← Slur 内可以有空格，但首尾必须紧贴括号
```

**Slur 嵌套：**

```abc
(D E F (G | G4))     ← Slur 内嵌套 Slur
(D E F G-| G4)       ← 等价写法，用 Tie
```

### 3.4 连音符（Tuplets）

| 写法          | 含义              |
| ----------- | --------------- |
| `(2GA`      | 二连音（2个音符占3个的时值） |
| `(3GAB`     | 三连音（3个音符占2个的时值） |
| `(4GABA`    | 四连音（4个音符占3个的时值） |
| `(5GABcd`   | 五连音             |
| `(6GABcded` | 六连音（占2个的时值）     |
| 依此类推        | 最多到 9           |

**完整语法 `(p:q:r)`：**

- `p` = 要放的音符数量
- `q` = 占多少拍的时值
- `r` = 持续这个动作的音符数量

```abc
(3:2:2G4c2          ← 2个音符占3拍，共2个音符
(3:2:4G2A2Bc        ← 4个音符按三连音分配
(3D2E2F2            ← 等价于 (3:2:3D2E2F2
```

**默认规则：**
- 如果不指定 `q`：复合拍号（3/4, 3/8, 9/8）默认为 3，简单拍号（C, 4/4, 2/4）默认为 2
- 如果不指定 `r`：等于 `p`

### 3.5 和弦与齐奏

**和弦用方括号 `[ ]`：**

```abc
[Acea]             ← A 小调和弦（四个音）
[GB][Ac] [B2d2]    ← 和弦连奏
[AA]               ← 齐奏（同音高，软件会显示上下两个符干）
```

> 约定：和弦内音符按**升序**排列。

### 3.6 伴奏和弦标记

用双引号 `" "` 表示吉他等伴奏和弦，放在对应音符之前：

```abc
"G"GB d2 | "D"DF A2 | "Am" c2e2 | "D7" d2f2
```

**和弦格式：** `note` + `accidental` + `type` + `/bass`

| 部分   | 可选值                                  | 示例                 |
| ---- | ------------------------------------ | ------------------ |
| 根音   | A-G                                  | `"G"`              |
| 升降号  | # 或 b                                | `"Bb"`, `"F#"`     |
| 和弦类型 | m, min, maj, sus, dim, +, 7, 9, 11 等 | `"Dm7"`, `"Gsus4"` |
| 低音   | /后跟低音                                | `"C/E"`, `"D7/F#"` |

> 注意：旧版 ABC 使用 `+Gm+` 格式，现已改为 `"Gm"`，请使用新格式。

### 3.7 换行与行尾

| 符号 | 含义 |
|------|------|
| `\` | 续行符——告诉软件这行和下一行属于同一行谱表 |
| `!` | 强制换行（某些软件支持，非正式标准）|

```abc
D \
|: "G"G2B B>AB | "C"c2A A>ce |
```

### 3.8 弓法标记

小提琴弓法用字母 `u`（上弓）和 `v`（下弓）：

```abc
v_Au=Bv^c          ← 下弓Ab, 上弓=B, 下弓^c
```

### 3.9 重音与断奏

| 符号 | 含义 |
|------|------|
| `.d` | 断奏（Staccato）——音符下方/上方加点 |
| `Td` | 颤音（Trill）标记 |
| `Hd` | 延长号（Fermata/Hold） |
| `Md` | 强调（Emphasis）——水平线 |

```abc
.G.A._B.c .d2.e.d.^c    ← 断奏音群
```

## 五、结构与排版

### 5.1 乐曲结构

一个 ABC 文件由**多个乐曲**组成，每首乐曲包含：

```
[头部 Header]    ← 从 X: 开始到 K: 结束
[曲体 Body]      ← K: 之后的音符内容
[空行]           ← 分隔下一首曲子
```

**规则：**
1. 每首曲子的第一个字段**必须**是 `X:`
2. 每个字段独占一行
3. 头部最后一个字段**必须**是 `K:`
4. 曲子之间用空行分隔
5. 曲子内部**不能**有空行（人类能读懂，但软件会出错）

### 5.2 段落（Parts）

用 `P:` 字段标注段落：

```abc
X:1
T:Speed The Plough
M:4/4
L:1/8
K:G
P:A
GABc dedB | dedB dedB |
P:B
g2g2 g4 | g2fe dBGB |
```

**头部声明段落顺序：**

```abc
P:AABCBD          ← 演奏顺序
P:(A2B2)2         ← AABBAABB 的简写
P:A2B2.(CD)3      ← 可读性更好的写法
```

### 5.3 歌词对齐

**方法一：整段歌词（W: 字段）**

```abc
W:How much is that doggie in the window
W:The one with the waggly tail?
```

这会作为文本块显示在乐谱下方。

**方法二：逐音符对齐（w: 字段）**

```abc
C D E F G A B c
w: Do - re mi fa sol la si do
```

每个音节对应一个音符，`-` 表示延长音，`_` 表示跳过音符。

### 5.4 注释与评论

用 `%` 符号添加注释——该符号之后直到行尾的内容都会被忽略：

```abc
% 这是文件级别的注释
X:1
T:Example
K:C
CDEFGABc  % 这是一首音阶练习
```

某些软件使用 `%%` 作为软件专用指令。

## 六、多声部与复调

ABC 支持多声部（polyphony）记谱，使用 `V:` 字段：

```abc
X:1
T:多声部示例
M:4/4
L:1/4
K:C
V:1 name="高声部" clef=treble
V:2 name="低声部" clef=bass
[V:1] C E G c
[V:2] C, E, G, C
```

- `V:` 定义声部，可指定名称、谱号等
- 用 `[V:1]` 在曲体中切换声部
- 也可用 `&` 符号在同一行内写多声部

## 七、实用参考

### 7.1 完整工作示例：Paddy O'Rafferty

这是 Chris Walshaw 博客中的经典教学示例，包含了之前讲到的所有基础知识：

```abc
X:1
T:Paddy O'Rafferty
C:Trad.
M:6/8
K:D
dff cee|def gfe|dff cee|dfe dBA|
dff cee|def gfe|faf gfe|1 dfe dBA:|2 dfe dcB|]
~A3 B3|gfe fdB|AFA B2c|dfe dcB|
~A3 ~B3|efe efg|faf gfe|1 dfe dcB:|2 dfe dBA|]
fAA eAA|def gfe|fAA eAA|dfe dBA|
fAA eAA|def gfe|faf gfe|dfe dBA:|
```

**逐行解析：**

| 行                    | 含义                              |
| -------------------- | ------------------------------- |
| `X:1`                | 索引编号 1                          |
| `T:Paddy O'Rafferty` | 曲名                              |
| `C:Trad.`            | 传统曲目（作曲者不可考）                    |
| `M:6/8`              | 六八拍（Jig 舞曲）                     |
| `K:D`                | D 大调（两个升号：F#、C#）                |
| `dff cee` …          | 音符内容，每小节用竖线分隔                   |
| `~A3`                | A 加装饰音（转音/滚奏），时值为 3 倍默认（附点四分音符） |
| `B2c`                | B 为两倍时值（四分音符），接默认时值的 c          |
| 第一结尾、第二结尾            | 见上方 §3.1 小节线与反复记号               |

> **关于 X: 字段**：有些人选择每首曲子单独存一个文件，因此可能遇到没有 X: 字段的曲子。但并非所有 ABC 软件都能识别这种情况，所以**建议始终包含 X: 字段**，即使只用 `X:1`。

**头部字段顺序规则：**
1. **第一个字段必须是 `X:`**
2. **之后必须有 `T:`**
3. **最后一个字段必须是 `K:`**（标记头部结束，音符开始）
4. 中间的其他字段（C、M、R 等）可以任意顺序排列

### 7.2 音束（Beams）

将两个或多个八分音符或十六分音符组合在一起形成音束，只需将它们紧挨着写，中间不加空格：

```abc
M:C
K:C
A B c d AB cd|ABcd ABc2|]
```

- `A B c d` — 空格分隔，显示为独立的音符
- `AB cd` — 无空格，形成音束连接
- `ABcd` — 四个音符连成一个音束

> 注意：音束只是为了视觉排版，不影响演奏效果。合理运用音束可以大幅提升乐谱的可读性。

### 7.3 常见曲种的默认时值

| 曲种 | 推荐 `L:` |
|------|-----------|
| Jig | `1/8` |
| Reel | `1/8` |
| Schottische | `1/8` |
| Waltz | `1/4` |
| Polka | `1/8` |
| Bourree | `1/8` |
| Hornpipe | `1/16` 或 `1/8` + `>` |

> 注意：这不是硬性规定。如果换一种时值写起来更清晰，完全可以使用不同的 `L:`。

### 7.4 符号顺序规则

当一个音符同时使用多种修饰符时，**正确顺序**为：

```
和弦标记 → 装饰符号 → 升降号 → 音名 → 八度标记 → 时值修饰
```

示例：

```abc
"Gm7"v.=G,2
```

| 部分 | 含义 |
|------|------|
| `"Gm7"` | 和弦标记 |
| `v.` | 下弓 + 断奏 |
| `=` | 还原号 |
| `G` | 音名 |
| `,` | 低八度 |
| `2` | 两倍时值 |

### 头部字段语法

每个头部字段格式为：`字母: 内容`

### 曲体中的特殊符号汇总

| 符号 | 含义 |
|------|------|
| `A-G, a-g` | 音符 |
| `, '` | 八度调整 |
| `^ ^^ _ __ =` | 升降还原号 |
| `1-9 /` | 时值修改 |
| `z` | 休止符 |

小节线与反复记号（含 `|` 字符，无法在表格中展示）：
- **`|`** 普通小节线
- **`||`** 双小节线
- **`[|`** / **`|]`** 粗-细 / 细-粗双小节线
- **`|:`** / **`:|`** 反复开始 / 反复结束
- **`::`** 反复开始和结束连写
- **`|1`** / **`|2`** 不同结尾

| 符号 | 含义 |
|------|------|
| `()` | 圆滑线 |
| `-` | 连音线 |
| `[]` | 和弦 |
| `{}` | 装饰音/倚音 |
| `~` | 转音/滚奏 |
| `> <` | 节奏倾斜 |
| `" "` | 伴奏和弦 |
| `%` | 注释 |
| `\` | 续行符 |
| `.` | 断奏 |
| `u v` | 弓法 |

### 关于 < 和 > 在 HTML 中的注意事项

在网页中嵌入 ABC 时，`<` 和 `>` 会被误认为 HTML 标签。正确做法：
- 将 ABC 放在独立的 `.abc` 文件中，通过链接引用
- 必须嵌入 HTML 时使用 `<pre>` 标签，并将 `<` 写为 `&lt;`、`>` 写为 `&gt;`
